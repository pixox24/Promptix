# Promptix 黑白 Logo 四宫格设计说明

## 项目语义

Promptix 是面向 AI 图像生成场景的提示词模板发现、编辑与生成工具。Logo 需要同时传达“提示词工具”“创意灵感”和轻微怪趣感，并延续现有品牌偏俏皮、友好的气质。

## 输出目标

- 生成一张正方形四宫格提案图，供用户选择后续深化方向。
- 四格均为原创黑白图标，不包含品牌文字、字母标签或说明文字。
- 采用扁平、矢量友好的造型：粗线条、清晰轮廓、平衡留白。
- 不使用颜色、灰阶、渐变、阴影、3D、拟物材质、展示样机或水印。
- 图标缩小到 favicon 尺寸时，核心轮廓仍可辨认。

## 四个方向

1. 左上｜提示词小怪兽：方形对话框与俏皮表情融合，像一只会吃 Prompt 的小怪物；突出提示、对话和怪趣角色感。
2. 右上｜灵感弹簧：将抽象字母 P、弹簧动作和一枚火花融合；突出灵感突然弹出的瞬间。
3. 左下｜括号笑脸：用一对代码花括号组成略带坏笑的脸；突出结构化提示词、工具属性和冷幽默。
4. 右下｜光标投喂：鼠标光标把一个小字符或 token 投进极简机器人嘴里；突出“输入提示，生成创意”的产品行为。

## 统一视觉系统

- 纯黑图形置于纯白背景。
- 四格等大，使用细黑分隔线；每格一个居中图标，四周保留充足安全边距。
- 四个图标使用近似一致的视觉重量、圆角语言和线条粗细。
- 幽默来自表情、姿态和视觉双关，不依赖复杂细节。
- 禁止出现任何可读文字、额外装饰、重复图标或现有品牌商标。

## 验收标准

- 一眼可见为严格 2×2 四宫格，并且四格内容互不重复。
- 在黑白条件下有强轮廓和良好正负形关系。
- 四个方向分别覆盖工具感、灵感感、代码感和交互感，同时保持同一家族气质。
- 画面干净，无颜色、灰度渲染、阴影、纹理、3D 和水印。
- 用户可以明确用“左上、右上、左下、右下”选择方案。
